## Purpose

AURA Safe Runtime Configuration Profile Foundation prepares safe runtime configuration profile blueprints for future runtime-readiness work.

This sprint is planner-only, metadata-only, configuration-blueprint-only, foundation-only, safe_idle-first, permission-first, and side-effect-free.
